# 🤖 AI Placement Preparation Agent

A simple web application that helps students prepare for placements.

## Features

- 👤 Personalized user setup
- 🧮 Aptitude practice
- 💻 Technical questions
- ⌨️ Coding practice
- 🎤 Mock interview
- 📊 Progress tracking
- 💬 AI Career Coach

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage

## How to Use

1. Enter your name.
2. Select your target role.
3. Select your skills.
4. Start your placement preparation.
5. Practice and track your progress.
